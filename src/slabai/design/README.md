# Design Folder

- `screens/`: normalized screen references used by specifications.
- `assets/`: currently supplied brand assets.
- `system/`: visual Design System and component-board references.
- `source/`: original Figma, product-flow PDF, core reference, and source-quality artifacts.
- `design-tokens.json`: framework-neutral tokens.
- `tokens.css`: CSS variables.
- `tailwind-theme.ts`: Tailwind theme fragment.
- `screen-map.json`: machine-readable mapping from reference to route.

Do not ship `design/source/` in a production bundle.
