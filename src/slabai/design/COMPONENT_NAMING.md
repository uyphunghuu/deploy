# Recommended Figma component naming

## Buttons
- Button / Primary / Size=MD / State=Default
- Button / Primary / Size=MD / State=Hover
- Button / Primary / Size=MD / State=Pressed
- Button / Primary / Size=MD / State=Disabled
- Button / Secondary / Size=MD / State=Default
- Button / Ghost / Size=MD / State=Default
- Button / Destructive / Size=MD / State=Default
- Button / Icon / Size=MD / State=Default

Variant properties:
- Type: Primary | Secondary | Ghost | Destructive | Icon
- Size: SM | MD | LG
- State: Default | Hover | Pressed | Disabled | Loading
- Icon: None | Leading | Trailing | Only
